Introduo

Quando realizamos um teste de hiptese, duas situaes de erro podem acontecer. fundamental entender esses erros para interpretar corretamente os resultados estatsticos.

O que Erro Tipo I?

Ocorre quando rejeitamos a hiptese nula, mesmo ela sendo verdadeira. chamado de "falso positivo".

Em um julgamento, a hiptese nula (H0) que o ru inocente. Se o tribunal rejeita H0 e condena o ru, mesmo que ele seja inocente, cometeu um Erro Tipo I (falso positivo condenar um inocente).

Exemplo na sade: Um exame de diagnstico que indica que um paciente tem uma doena quando na verdade ele est saudvel.

Qual a relao do Erro tipo I e o nvel de significncia ()?

A probabilidade de cometer um erro Tipo I (isto , rejeitar a hiptese nula quando ela verdadeira) definida pelo nvel de significncia . Em outras palavras: Quando voc escolhe = 0,05, voc est aceitando que, mesmo que a hiptese nula seja verdadeira, voc vai cometer um erro Tipo I em 5% dos casos. Ento, uma escolha feita antes de olhar os dados uma definio prvia de risco que voc aceita como tolervel para o seu estudo.

Mas cuidado! O que muita gente confunde :

  • definido antes do estudo.
  • p-valor calculado depois dos dados serem coletados.

O p-valor ento comparado ao para decidir:

  • Se p < rejeitamos a hiptese nula.
  • Se p no rejeitamos a hiptese nula.

" define o risco que aceitamos cometer ao rejeitar H. O p-valor o que medimos depois, para ver se cruzamos esse limite."

O que Erro Tipo II?

Ocorre quando no rejeitamos a hiptese nula, mesmo ela sendo falsa. chamado de "falso negativo". O Erro Tipo II representado pela letra grega (beta).

Em um julgamento, a hiptese nula que o ru inocente. Se o tribunal no rejeita H0 (ou seja, no condena) e o ru de fato culpado, cometeu um Erro Tipo II (falso negativo absolver um culpado).

Exemplo na sade: Um exame que indica que o paciente est saudvel quando ele realmente tem uma doena.

Qual a relao do Erro tipo II e o poder estatstico?

Assim como definimos o nvel de significncia () antes de coletar os dados, tambm definimos o valor de aceitvel antes, durante o planejamento do estudo. Quando desenhamos um estudo (por exemplo, um ensaio clnico ou uma pesquisa estatstica), ns planejamos:

  • = quanto risco aceitamos cometer um erro Tipo I (geralmente adotado como: 5% ou 0,05).
  • = quanto risco aceitamos cometer um erro Tipo II (geralmente adotadoo como: 20% ou 0,20).

O poder estatstico de um teste a probabilidade de detectar um efeito real quando ele realmente existe. Ele oo complementar de Beta.

Ento:

  • Quanto maior o Erro Tipo II () menor o Poder Estatstico (1 - ).
  • Quanto menor o Erro Tipo II () maior o Poder Estatstico (1 - ).

Por que o poder estatstico importa?

  • Se o poder estatstico for baixo, mesmo que o efeito exista, seu estudo pode no detect-lo.

  • Estudos com alto poder so mais confiveis: se houver um efeito real, eles tm grande chance de encontr-lo.

Visualizando o Erro tipo I e o Erroo tipo II

import numpy as np
import matplotlib.pyplot as plt
from scipy.stats import norm

# Parmetros da distribuio sob H0 (sem efeito)
mu_H0 = 0
sigma = 1

# Definindo o limiar de rejeio para alfa = 0.05 (teste bilateral)
alpha = 0.05
z_alpha_half = norm.ppf(1 - alpha/2)  # valor crtico para 2,5% de cada lado

# Criar valores no eixo x
x = np.linspace(-4, 6, 1000)
y_H0 = norm.pdf(x, mu_H0, sigma)

# Parmetros para H1 (efeito real ajustado para poder de 80%)
mu_H1 = 2.8
y_H1 = norm.pdf(x, mu_H1, sigma)

# Criando o grfico
plt.figure(figsize=(10,6))
plt.plot(x, y_H0, label="Distribuio sob H0 (sem efeito)")
plt.plot(x, y_H1, label="Distribuio sob H1 (efeito verdadeiro)", linestyle='--')

# Sombrear reas de Erro Tipo I (alfa)
plt.fill_between(x, y_H0, where=(x > z_alpha_half) | (x < -z_alpha_half), color='red', alpha=0.3, label="Erro Tipo I ()")

# Sombrear reas de Erro Tipo II (beta)
plt.fill_between(x, y_H1, where=(x > -z_alpha_half) & (x < z_alpha_half), color='blue', alpha=0.3, label="Erro Tipo II ()")

# Marcar os pontos crticos
plt.axvline(z_alpha_half, color='black', linestyle='--')
plt.axvline(-z_alpha_half, color='black', linestyle='--')

# Configuraes
plt.title('Erro Tipo I () e Erro Tipo II () - Poder Estatstico 80%')
plt.xlabel('Estatstica de Teste')
plt.ylabel('Densidade de Probabilidade')
plt.legend()
plt.grid(True)
plt.show()

As duas curvas representam o comportamento da estatstica de teste quando no h efeito e quando h efeito, respectivamente.

  • As pontas vermelhas (fora dos limites de corte) mostram onde rejeitamos H0 mesmo que H0 fosse verdadeira isso Erro Tipo I ().
  • A rea azul mostra onde no conseguimos rejeitar H0 apesar de H1 ser verdadeira isso Erro Tipo II ( ).
  • O poder estatstico do teste a rea fora da regio azul, sob a curva laranja H1.
