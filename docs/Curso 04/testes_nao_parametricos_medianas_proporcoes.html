📌 Testes Não Paramétricos para Medianas e Proporções¶

⚖️ Introdução¶

  • Testes não paramétricos são usados quando:
  • Os dados não seguem distribuição normal (falha em testes como Shapiro-Wilk).
  • São ordinais (ex.: escala de dor) ou nominais (ex.: grupos sanguíneos).
  • Contêm outliers que distorcem análises paramétricas.

🔍 Parte 1: Testes Não Paramétricos para Medianas¶

📌 Teste de Mann-Whitney (U)¶

  • Quando usar? Alternativa ao teste t para amostras independentes quando os dados não são normais.

  • Compara duas medianas de grupos não relacionados.

  • Exemplo em Saúde: Comparar a eficácia de dois analgésicos (A vs. B) em escalas de dor (0-10).

In [ ]:
from scipy.stats import mannwhitneyu

grupo_A = [3, 5, 2, 6, 4]  # Analgésico A
grupo_B = [7, 8, 6, 9, 7]  # Analgésico B

stat, p = mannwhitneyu(grupo_A, grupo_B)
print(f"p-valor = {p:.4f}")  # p ≤ 0.05 → diferença significativa

p-valor = 0.0157

📝 Interpretação:

  • H₀: As medianas são iguais.
  • p ≤ 0.05 → Rejeita H₀ (há diferença entre os grupos).

📌 Teste de Wilcoxon Signed-Rank¶

  • Quando usar? Alternativa ao teste t pareado para dados não normais.
  • Compara medianas antes/depois no mesmo grupo.
  • Exemplo em Saúde: Avaliar redução de pressão arterial após um tratamento.
In [ ]:
from scipy.stats import wilcoxon

antes = [140, 150, 130, 145, 135]
depois = [130, 140, 125, 135, 130]

stat, p = wilcoxon(antes, depois)
print(f"p-valor = {p:.4f}")  # p ≤ 0.05 → efeito significativo

p-valor = 0.0625

📝 Interpretação:

  • H₀: Não há diferença entre as medições pareadas.
  • p ≤ 0.05 → O tratamento teve efeito.

📌 Teste de Kruskal-Wallis¶

  • Quando usar? Alternativa à ANOVA para 3+ grupos independentes.
  • Verifica se pelo menos um grupo tem mediana diferente (Compara distribuições de ranks ou ordenações para inferir sobre diferenças nas medianas ou formas das distribuições).
  • Exemplo em Saúde: Comparar tempo de recuperação em três hospitais.

😊 Testes post-hoc Dunn (Bonferroni)¶

Dunn's é para dados não paramétricos (baseado em rankings). Tukey HSD é para dados paramétricos (baseado em médias).

Regra geral:

  • Se usou Kruskal-Wallis → Dunn (Bonferroni)
  • Se usou ANOVA → Tukey HSD.

🔍 Parte 2: Testes Não Paramétricos para Proporções¶

📌 Teste Qui-Quadrado (χ²)¶

  • Quando usar? Comparar proporções em tabelas de contingência. O teste qui-quadrado de Pearson compara frequências observadas com frequências esperadas sob a hipótese nula (H₀) de independência entre variáveis categóricas.

  • Exemplo: Verificar se há associação entre fumante (Sim/Não) e doença pulmonar (Sim/Não).

In [ ]:

Cálculo manual¶

Calcule os valores esperados (se H₀ fosse verdadeira):

  • Os valores esperados refletem a distribuição que ocorreria se não houvesse relação entre fumar e ter a doença.
  • Usa a distribuição marginal das linhas e colunas
  • Fórmula para célula (i,j):

Sob a hipótese nula (H₀) de não associação entre fumo e doença:

  • A proporção geral de doença é 40/200 = 20%
  • Esperaríamos que 20% dos fumantes (20 de 100) e 20% dos não-fumantes (20 de 100) tivessem doença
  • O mesmo raciocínio se aplica aos saudáveis (80 em cada grupo)

Comparando observado vs esperado:

  • Fumantes com doença: 30 (observado) vs 20 (esperado) → +50% mais casos
  • Não-fumantes com doença: 10 vs 20 → 50% menos casos
  • Essas discrepâncias são o que o teste qui-quadrado avalia para determinar se a associação é estatisticamente significativa

Calcule a estatística χ²:

É uma distribuição teórica que: Descreve como varia a soma de quadrados de variáveis normais padrão, tem forma assimétrica à direita e depende dos graus de liberdade (gl)

📌 Teste exato de Fisher¶

  • Quando usar? Use o teste exato de Fisher quando você deseja avaliar a associação entre duas variáveis categóricas em uma tabela 2x2 (duas linhas e duas colunas), especialmente quando os valores esperados são pequenos (menores que 5). Diferente do teste qui-quadrado, não depende de aproximações. É ideal para amostras pequenas.
  • Exemplo: Vamos verificar se existe associação entre ser fumante e ter doença pulmonar com uma amostra pequena:
In [3]:

import pandas as pd

# tabela 2x2
# linha 1: fumantes, linha 2: não fumantes
# colunas: com doença, sem doença

O Teste de Fisher calcula todas as possíveis combinações de tabelas 2x2 com os mesmos totais marginais, assumindo que a hipótese nula (independência entre as variáveis) é verdadeira.

➡️ O p-valor obtido é a soma das probabilidades dessas tabelas tão extremas quanto ou mais do que a observada.

Por isso, é considerado exato, sem aproximações da distribuição qui-quadrado.

In [5]:
oddsratio, p_valor = stats.fisher_exact(tabela)
print(f"p-valor = {p_valor:.4f}")

📝 Interpretação:

  • Se p ≤ 0,05, rejeita-se H₀ ➔ há associação significativa.
  • Se p > 0,05, não se rejeita H₀ ➔ não há evidência de associação.

No exemplo, o p-valor indica que existe associação significativa entre ser fumante e ter doença pulmonar.
